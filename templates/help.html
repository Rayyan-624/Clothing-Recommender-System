Frequently Asked Questions

How does the recommendation system work?
The system uses a hybrid approach combining rule-based logic and machine learning. It analyzes three main factors: temperature compatibility (40% weight), weather condition matching (30% weight), and occasion formality (30% weight). Each clothing item is scored, and the system builds complete outfits by selecting compatible items from different categories.
What weather conditions are supported?
The system supports 8 weather types: Clear/Sunny, Cloudy, Rainy, Snowy, Windy, Partly Cloudy, Thunderstorm, and Foggy. Each weather type is mapped to appropriate clothing characteristics for optimal recommendations.
What occasions can I select?
Available occasions include: Casual, Business, Formal, Sports, Date, Outdoor, Beach, and Party. Each occasion has an associated formality level that influences the clothing recommendations.
How many outfit recommendations will I get?
The web interface provides 3 complete outfit recommendations, ranked by suitability score. The API endpoint can be configured to return up to 3 recommendations. Each outfit includes items from multiple categories (Top, Bottom, Footwear, and optionally Outerwear or Accessories).
What does the score mean?
The score (0-100) represents how well the outfit matches your specified conditions. Higher scores indicate better matches. Scores above 70 are excellent, 50-70 are good, and below 50 suggest the outfit may not be ideal for the conditions.
Can I use this programmatically?
Yes! We provide a REST API at /api/recommend. See the API Documentation for details on how to integrate the recommendation system into your own applications.
How is the dataset built?
The system uses a curated dataset of 90+ clothing items, each with detailed attributes including warmth level, formality, suitable temperature ranges, weather tags, occasion tags, colors, and materials. The dataset covers all major clothing categories.

Tips for Best Results

  • Enter the actual temperature you'll be experiencing, not the "feels like" temperature
  • Select the most specific weather condition - if it's both cloudy and rainy, choose rainy
  • For mixed occasions (e.g., business-casual), start with the more formal option
  • Use the Quick Settings buttons for common scenarios to see how the system works
  • Review multiple recommendations - you might prefer a lower-scored outfit based on personal style
  • The system considers layering - warmer outfits may include outerwear for temperature control
