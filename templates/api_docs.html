API Documentation

RESTful API for clothing recommendations

Base URL

http://127.0.0.1:5000

GET Get Outfit Recommendations

GET /api/recommend

Returns clothing outfit recommendations based on weather conditions and occasion.

Query Parameters

Parameter Type Description Example
temp optional float Temperature in Celsius (default: 20) 25
weather optional string Weather condition (default: 'clear') sunny, cloudy, rainy, snowy
occasion optional string Occasion type (default: 'casual') casual, business, formal, sports

Example Request

curl "http://127.0.0.1:5000/api/recommend?temp=25&weather=sunny&occasion=casual"

Example Response

{ "success": true, "recommendations": [ { "avg_score": 94.0, "categories": ["Top", "Bottom", "Footwear", "Accessory"], "pieces": [ { "item_id": 1, "item_name": "Cotton T-Shirt", "category": "Top", "warmth_level": 1, "formality": 2, "min_temp": 17, "max_temp": 31, "weather_tags": "hot,warm", "occasion_tags": "casual,sports", "color": "Blue", "material": "Cotton" }, ... ], "total_score": 376.0, "weather_type": "sunny" } ] }

Response Fields

Field Type Description
success boolean Whether the request was successful
recommendations array List of outfit recommendations (up to 2)
avg_score float Average suitability score (0-100)
categories array Clothing categories in the outfit
pieces array Individual clothing items in the outfit

Error Responses

If an error occurs, the API returns:

{ "success": false, "error": "Error message description" }

Usage Examples

JavaScript (Fetch API)

fetch('http://127.0.0.1:5000/api/recommend?temp=20&weather=cloudy&occasion=business') .then(response => response.json()) .then(data => { console.log(data.recommendations); }) .catch(error => console.error('Error:', error));

Python (Requests)

import requests params = { 'temp': 20, 'weather': 'cloudy', 'occasion': 'business' } response = requests.get('http://127.0.0.1:5000/api/recommend', params=params) data = response.json() if data['success']: for outfit in data['recommendations']: print(f"Score: {outfit['avg_score']}") for item in outfit['pieces']: print(f" - {item['category']}: {item['item_name']}")
